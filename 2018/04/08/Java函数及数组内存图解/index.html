数组

Java数组基础

内存

  • 内存的划分
    • 寄存器
    • 本地方法区
    • 方法区
    • 栈内存
    • 堆内存

栈内存

  • 存储的都是局部变量，变量所属作用域一旦结束，该变量就自动释放

堆内存

  • 储存数组和对象（数组就是对象），new 建立的都在堆上
  • 特点：
    1. 每个实体都有首地址值
    2. 堆内存中每个变量都有默认内存初始值，根据类型不同而不同
      （int -> 0,double -> 0.0,float -> 0.0f,boolean -> false,char -> ‘\u0000’）
    3. 垃圾回收机制
