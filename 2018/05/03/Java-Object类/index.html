发表于 2018-05-03 | 分类于 JavaSE | 阅读次数:

Object类

  • Object ：所有类的根类
  • Object 是抽取而来的，具备着所有对象都具备的共性内容
  • 常用的共性内容：
    • equals 方法
    • hashCode 方法
    • getClass 方法
    • toString 方法

      equals方法

  • equals 实现对象上差别可能性最大的相等关系，即，对于任何非空引用值 x 和 y ，当且仅当 x 和 y引用同一个对象时，此方法才返回 true （x==y 具有值 true）

  • equals 方法重写，比较成员变量值
