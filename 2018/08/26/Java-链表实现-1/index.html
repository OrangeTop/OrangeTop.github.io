什么是链表

链表 [Linked List]：链表是由一组不必相连不必相连：可以连续也可以不连续的内存结构 节点，按特定的顺序链接在一起的抽象数据类型。

补充：
抽象数据类型（Abstract Data Type [ADT]）：表示数学中抽象出来的一些操作的集合。
内存结构：内存中的结构，如：struct、特殊内存块…等等之类；

  • 最简单的数据结构
  • 更深入的理解引用，指针
  • 更深入的理解递归
  • 辅助组成其他数据结构

优缺点

  • 优点：真正的动态，不需要处理固定容量的问题
  • 缺点：丧失了随机访问的能力
