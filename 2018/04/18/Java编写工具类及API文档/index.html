报错

编码GBK的不可映射字符

  • 解决方法
    • 关闭编译器要编写的那个工具类
    • 用记事本打开 Java 源文件
    • 另存为，选择 ANSI 编码
    • 覆盖后，在DOS界面重试
    • 成功后，将编码改回原格式

错误2

  • 生成错误原因：改为ANSI编码后，打开了编译器，汉字都变成了乱码
  • 切记，记事本改完编码后，不要。立即用编辑器打开该文件
